# BMI Calculator App

A simple Android BMI (Body Mass Index) Calculator app built with Kotlin and XML layouts.

## Features

- Calculate BMI from weight (kg) and height (cm)
- Display BMI result and category (Underweight, Normal Weight, Overweight, Obese)
- Simple, beginner-friendly interface

## Requirements

- Android Studio Hedgehog (2023.1.1) or later
- Min SDK: 21 (Android 5.0)
- Target SDK: 34

## Building the Project

1. Open the project in Android Studio
2. Sync Gradle files
3. Run the app on an emulator or physical device

## Usage

1. Enter your weight in kilograms
2. Enter your height in centimeters
3. Tap "Calculate BMI" button
4. View your BMI result and category
